# ensinando python 😥

repositório com todo o conteúdo utilizado durante as aulas, sempre atualizado.

## estrutura

cada módulo está organizado em sua própria pasta com seguinte estrutura padrão:
* **`arquivo.pdf`**: os slides utilizados na aula.
* **`códigos/`**: scripts que demonstram funcionalidades do conteúdo visto no slide.
* **`exercicios/`**: resolução de exercícios (quando disponíveis).
* **`exemplos/`**: implementação em script dos exemplos do slide.
